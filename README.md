# PBR-Study

PBR-Study は、DirectX 9 ベースで Physically Based Rendering の要素を段階的に試すための検証用リポジトリである。現時点では `basic/simple-directx9` にシンプルなサンプルがあり、ベースカラー、ラフネス、メタリック、環境反射の挙動を確認できる構成になっている。

## 概要

このサンプルは、固定機能寄りの時代の API である DirectX 9 上に、PBR らしい見た目をどこまで持ち込めるかを試すことを目的としている。`simple.fx` では Cook-Torrance 系のスペキュラ計算、GGX 分布、Schlick の近似式を用いたフレネル項、簡易的な環境反射を扱っており、学習用途として読みやすい形にまとめられている。

実行時にはモデルを表示しつつ、PBR パラメータや環境反射の強さを画面上に表示する。`F1` からモデルや各種値を調整するダイアログを開き、見た目の変化をその場で確認できる。

## ディレクトリ構成

- `basic/simple-directx9`
  DirectX 9 で動作するサンプル本体である。
- `basic/simple-directx9/main.cpp`
  アプリケーション本体、カメラ移動、UI、モデル読み込み、描画制御を担当する。
- `basic/simple-directx9/simple.fx`
  PBR 風ライティングと簡易キューブマップ反射を記述したエフェクトファイルである。
- `basic/simple-directx9/*.x`
  表示に用いるメッシュである。
- `basic/simple-directx9/*.png`, `basic/simple-directx9/*.dds`
  背景や環境反射に使用するテクスチャ群である。

## ビルド環境

このリポジトリには `basic/simple-directx9/simple-directx9.sln` が含まれている。Visual Studio 2022 系のツールセット `v143` を前提としており、プロジェクトでは NuGet パッケージ `Microsoft.DXSDK.D3DX` を参照している。

ビルド前には NuGet パッケージの復元が必要である。ソリューションを Visual Studio で開いて復元してもよいし、MSBuild から復元とビルドを実行してもよい。

## 実行時の操作

- `W`, `A`, `S`, `D`
  カメラを前後左右に移動する。
- `E`, `Q`
  カメラを上下に移動する。
- `Esc`
  カーソル表示とマウスルックを切り替える。
- `F1`
  モデル選択および PBR パラメータ調整用ダイアログを開く。

## このリポジトリの位置づけ

本リポジトリは完成品のレンダラというより、描画式やパラメータの効き方を確認するための実験場である。DirectX 9 という制約の中で、現代的な PBR の考え方をどのように近似しているかを追いやすいため、シェーダ学習や既存描画コードの比較検証に向いている。
